# Chat Backend

Node.js + Express + MongoDB API. Use this as its own repo.

## Setup

```bash
npm install
cp .env.example .env
# Edit .env: set MONGODB_URI if needed
```

## Run

```bash
npm run dev
```

Runs on http://localhost:5000. Health: `GET /api/health`.

## Repo

Push this folder to its own Git repo (e.g. `chat-backend` on GitHub).
